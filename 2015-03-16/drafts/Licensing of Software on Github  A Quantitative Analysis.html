What is gitPAN?

gitPAN is a project to import the entire history of CPAN (known as BackPAN) into a set of git repositories, one per distribution.

What is CPAN?

CPAN is the Comprehensive Perl Archive Network at cpan.org. It is an archive of tens of thousands of Perl modules written by thousands of authors. A good interface to it is http://search.cpan.org

Interpreting the data
